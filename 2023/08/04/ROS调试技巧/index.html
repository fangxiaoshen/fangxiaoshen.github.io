差速运动学模型：

两轮中心:线速度 = (右轮速度+ 左轮速度) / 2：v = (vr+ vl) / 2

由 右轮速度/角速度 - 左轮速度/角速度 = 左轮到右轮的距离 , 得角速度 = (右轮速度- 左轮速度) / 左轮到右轮的距离：由 vr/w - vl/w = l , 得w = (vr- vl) / l

圆弧运动轨迹中心O到两轮中心距离: r = v / w = ( l * (vr+ vl) ) / ( 2 * (vr- vl) )
